- #### Exercise

- Write a program that tells whether someone is a kid, teenager, or adult based on their age.

**Example Solution:**
```javascript
let age = 9;
if (age < 13) {
  console.log("You are a kid.");
} else if (age < 20) {
  console.log("You are a teenager.");
} else {
  console.log("You are an adult.");
}
```
